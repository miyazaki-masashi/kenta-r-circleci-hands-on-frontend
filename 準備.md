## S3を用意する
バケット名を変更して作成する
```
# バケット名を環境変数に設定
export BUCKET_NAME=[作成するバケット名]
# S3バケットを作成する
aws s3 mb s3://$BUCKET_NAME
# 静的サイトホスティングを有効にする
aws s3api put-bucket-website --bucket $BUCKET_NAME --website-configuration file://XXXXXXXXX/website.json
# S3のアクセス制御をブロックなし→ブロックにする
aws s3api put-public-access-block --bucket $BUCKET_NAME --public-access-block-configuration  "BlockPublicAcls=true,IgnorePublicAcls=true,BlockPublicPolicy=true,RestrictPublicBuckets=true"
# S3のアクセス制御（ポリシー）を自宅から許可する設定にする
aws s3api put-bucket-policy --bucket $BUCKET_NAME --policy file://XXXXXXXXXbucket-policy.json
```

``` website.json
{
    "IndexDocument": {
        "Suffix": "index.html"
    },
    "ErrorDocument": {
        "Key": "error.html"
    }
}
```

